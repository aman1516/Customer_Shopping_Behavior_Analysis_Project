# Customer_Shopping_Behavior_Analysis_Project

🛒 Customer Shopping Behaviour Analysis

A complete Data Analytics Project using Python, SQL Server, and Power BI to uncover customer purchasing patterns, revenue trends, discount impact, and customer segmentation.

📌 Project Overview
This project analyzes 5,000+ customer shopping records from a retail/e-commerce dataset to help businesses:
Increase revenue
Improve customer retention
Optimize discount strategies
Understand customer behavior
Build data-driven marketing decisions

📌 Key Insights
💰 Revenue Insights
📦 Electronics generated the highest revenue ($486K+)
💳 Customers using discounts spent 20% more
👑 Subscribers spent 63% more than non-subscribers

👥 Customer Insights
🔁 61.7% customers are loyal buyers
🏆 New York generated the highest revenue
⭐ Gloves received the best ratings
📉 Phone category had the lowest average rating
